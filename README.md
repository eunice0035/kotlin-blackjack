# kotlin-blackjack
## 기능 요구 사항

### - 플레이어
- [ ] 각각의 플레이어는 현재 카드의 숫자 합이 21이상이면 카드를 뽑을 수 없다.
- [x] 갖고 있는 카드를 계산하는 기능
- [ ] 매 턴마다 카드를 저장한다.

### - 카드 덱
- [ ] 카드는 총 52장이다. 
- [ ] 드로우 할 때 중복된 카드가 나오지 않도록 한다.
- [ ] 게임을 시작할 때 덱에 카드를 채운다.

### - 입력
- [x] 플레이어의 이름을 한줄로 입력받는다
- [x] 플레이어의 이름은 쉼표(,)로 분리한다
- [x] 플레이어는 0명이 될 수 없다.
